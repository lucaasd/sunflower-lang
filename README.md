# Sunflower Lang

### A simple and easy-to-use object oriented programming language

## Install the compiler

### Building from source

#### Requirements

- Clang
- LLVM
- CMake
- Make

#### Steps

##### Make a build dir to build the compiler

```sh
mkdir build

```

##### Run CMake
```sh
cd build
cmake ..
```

##### Build the compiler
```
make
```
